Bibliothèque de Composants Marki-parse

Collection complète des composants Lit pour l'application Impayés

Composants de Layout

impayes-page-layout
Layout Principal

Composant de mise en page principal pour la page Impayés avec gestion des différents modes de vue.

// Usage <impayes-page-layout title="Impayés" description="Gérez vos impayés ici." currentView="grouped-by-payer" ></impayes-page-layout>

Composants de Cartes

impaye-card-list
Carte de Facture

Affiche les détails d'une facture individuelle avec actions.

// Usage <impaye-card-list .invoice="invoiceObject" ></impaye-card-list>
impaye-card-by-payeur
Carte Groupée par Payeur

Affiche les factures groupées par payeur avec liste expandable.

// Usage <impaye-card-by-payeur .payer="payerObject" .invoices="invoicesArray" ></impaye-card-by-payeur>
impaye-card-sequence
Carte de Séquence Kanban

Carte draggable pour l'organisation des séquences en mode Kanban.

// Usage <impaye-card-sequence .payer="payerObject" .invoices="invoicesArray" .sequence="sequenceObject" ></impaye-card-sequence>

Composants UI

impaye-search
Recherche de Factures

Composant de recherche avec gestion des événements de recherche.

// Usage <impaye-search placeholder="Rechercher..." @search-updated="handleSearch" ></impaye-search>
view-mode-toggle
Sélecteur de Mode de Vue

Permet de basculer entre différents modes de visualisation.

// Usage <view-mode-toggle .currentMode="grouped-by-payer" @view-mode-changed="handleViewChange" ></view-mode-toggle>
sequence-drawer
Tiroir de Gestion des Séquences

Tiroir modal pour l'assignation et la gestion des séquences.

// Usage <sequence-drawer .visible="true" .invoices="selectedInvoices" @sequence-assigned="handleAssignment" ></sequence-drawer>

Composants Utilitaires

loading-indicator
Indicateur de Chargement

Affiche un indicateur de chargement avec spinner et barre de progression.

// Usage <loading-indicator .visible="true" message="Chargement..." ></loading-indicator>
empty-state
État Vide

Affiche un message lorsque aucun contenu n'est disponible.

// Usage <empty-state visible message="Aucun élément" icon="file-invoice" ></empty-state>
error-message
Message d'Erreur

Affiche des messages d'erreur avec différents types (erreur, avertissement, info).

// Usage <error-message visible message="Erreur..." type="error" ></error-message>
